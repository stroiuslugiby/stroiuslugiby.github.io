# Stroiuslugi - Сайт строительной компании

Многостраничный сайт для строительной компании на Bootstrap 5, готовый для размещения на GitHub Pages.

## Структура проекта

```
/
├── index.html              # Главная страница
├── about.html              # О компании
├── contact.html            # Контакты
├── portfolio.html          # Портфолио
├── sitemap.xml             # Карта сайта для SEO
├── robots.txt              # Файл для поисковых систем
├── README.md               # Этот файл
├── css/
│   └── style.css           # Кастомные стили
├── js/
│   └── main.js             # Основной JavaScript
└── services/               # Страницы услуг
    ├── painting.html
    ├── rough_finish.html
    ├── flooring.html
    ├── tiling.html
    ├── construction.html
    ├── facade.html
    ├── paving.html
    ├── concrete.html
    └── roofing.html
└── img/                    # Изображения
    ├── painting/
    ├── rough_finish/
    ├── flooring/
    ├── tiling/
    ├── construction/
    ├── facade/
    ├── paving/
    ├── concrete/
    └── roofing/
```

## Особенности

- ✅ Полностью адаптивный дизайн (Bootstrap 5)
- ✅ SEO оптимизация (уникальные title, meta description, h1 для каждой страницы)
- ✅ Open Graph мета-теги для соцсетей
- ✅ Sitemap.xml для индексации
- ✅ Robots.txt для управления доступом поисковых систем
- ✅ Lightbox для галереи
- ✅ Модальные окна для форм заявок
- ✅ Валидация форм
- ✅ Фильтрация портфолио
- ✅ Аккордеон FAQ
- ✅ Карусель отзывов

## Установка и запуск

1. Клонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в браузере для локального просмотра
3. Для публикации на GitHub Pages:
   - Создайте репозиторий на GitHub
   - Загрузите все файлы
   - Включите GitHub Pages в настройках репозитория
   - Обновите URL в `sitemap.xml` и `robots.txt`
