# Архитектура проекта Stroiuslugi

## Общая структура проекта

Проект представляет собой многостраничный статический веб-сайт строительной компании, построенный на Bootstrap 5. Структура организована по принципу разделения контента по типам и назначению.

## Структура папок и файлов

```
stroiuslugii/
├── index.html                    # Главная страница сайта
├── about.html                    # Страница "О компании"
├── contact.html                  # Страница контактов
├── portfolio.html                # Страница портфолио с фильтрацией
├── README.md                     # Основная документация проекта
├── robots.txt                    # Файл для управления индексацией поисковыми системами
├── sitemap.xml                   # XML карта сайта для SEO
│
├── css/                          # Папка со стилями
│   └── style.css                # Основной файл кастомных стилей
│
├── js/                           # Папка со скриптами
│   └── main.js                  # Основной JavaScript файл с общей логикой
│
├── services/                     # Папка со страницами услуг
│   ├── painting.html            # Малярные работы
│   ├── rough_finish.html        # Черновая отделка
│   ├── flooring.html            # Финишные покрытия
│   ├── tiling.html              # Укладка плитки
│   ├── construction.html        # Строительные работы
│   ├── facade.html              # Фасадные работы
│   ├── paving.html              # Тротуарная плитка
│   ├── concrete.html            # Бетонные работы
│   ├── roofing.html             # Кровельные работы

    ├── construction/             # Изображения для строительных работ (24 файла .webp)

    ├── flooring/                # Изображения для финишных покрытий (13 файлов .webp)

    └── tiling/                  # Изображения для укладки плитки (43 файла .webp)
```

## Назначение папок

### Корневая директория
- **HTML файлы верхнего уровня**: Основные страницы сайта (главная, о компании, контакты, портфолио)
- **robots.txt**: Управляет доступом поисковых роботов к различным разделам сайта
- **sitemap.xml**: Содержит карту сайта для поисковых систем, включает все страницы с приоритетами и частотой обновления
- **README.md**: Документация проекта с инструкциями по установке и использованию

### css/
Содержит кастомные стили, которые дополняют Bootstrap 5:
- Определение CSS-переменных для цветовой схемы
- Стили для навигации, hero-секций, карточек услуг
- Стили для галереи, модальных окон, форм
- Адаптивные стили для мобильных устройств
- Анимации и эффекты наведения

### js/
Содержит JavaScript логику:
- **main.js** - основная логика:
  - Инициализация lightbox для галереи изображений
  - Валидация форм
  - Управление модальными окнами
  - Функции для открытия форм заявок
  - Обработка отправки форм (с заглушками для backend интеграции)
- **services-gallery.js** - модуль для галерей услуг:
  - Автоматическая инициализация галерей на страницах услуг
  - Lightbox с навигацией между изображениями
  - Поддержка клавиатурной навигации
  - Счетчик изображений
  - Полная документация в комментариях

### services/
Содержит 10 страниц услуг, каждая из которых:
- Имеет уникальную структуру с hero-секцией
- Содержит описание услуги, виды работ, преимущества
- Включает галерею примеров работ
- Содержит отзывы клиентов
- Имеет форму заявки в модальном окне
- Использует breadcrumbs для навигации

### img/
Организована по категориям услуг:
- Каждая подпапка соответствует определенному типу работ
- Изображения хранятся в формате WebP для оптимизации
- Названия файлов следуют паттерну: `{category}_stroiuslugi.by_{number}.webp`
- Некоторые папки (concrete, roofing) пустые или содержат меньше изображений

## Ключевые файлы

### index.html
Главная страница сайта, содержит:
- Навигационное меню с выпадающим списком услуг
- Hero-секцию с призывом к действию
- Секцию с карточками всех услуг
- Галерею выполненных работ
- FAQ секцию с аккордеоном
- Карусель отзывов
- Секцию преимуществ
- CTA секцию
- Футер с контактами и социальными сетями
- Модальное окно формы заявки

### css/style.css
Основной файл стилей, определяет:
- Цветовую палитру через CSS-переменные
- Типографику и базовые стили
- Компоненты: карточки, кнопки, формы, модальные окна
- Адаптивные breakpoints для мобильных устройств
- Анимации и переходы

### js/main.js
Основной JavaScript файл, реализует:
- Инициализацию lightbox для просмотра изображений
- Валидацию HTML5 форм
- Управление модальными окнами Bootstrap
- Обработку отправки форм заявок
- Вспомогательные функции для навигации

### services/*.html
Страницы услуг имеют единообразную структуру:
- Фиксированная навигация с активным пунктом меню
- Breadcrumbs для навигации
- Hero-секция с заголовком услуги
- Детальное описание услуги с видами работ
- Галерея примеров выполненных работ
- Секция преимуществ
- Отзывы клиентов
- CTA секция с призывом к действию
- Футер
- Модальное окно формы заявки

## Особенности архитектуры

1. **Статическая структура**: Проект не использует серверную логику, все страницы статические HTML
2. **CDN зависимости**: Bootstrap 5 и Bootstrap Icons загружаются через CDN
3. **Единообразие**: Все страницы услуг следуют единому шаблону для консистентности
4. **SEO оптимизация**: Каждая страница имеет уникальные meta-теги, title, description
5. **Адаптивность**: Использование Bootstrap grid system для responsive дизайна
6. **Модульность**: Стили и скрипты вынесены в отдельные файлы для переиспользования
