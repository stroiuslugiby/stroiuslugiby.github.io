## Разработка страниц услуг

### Создание шаблона страницы услуги
- Разработан единообразный шаблон для всех страниц услуг
- Реализована структура: hero → описание → галерея → преимущества → отзывы → CTA
- Добавлены breadcrumbs для навигации
- Создана боковая карточка "Быстрый расчет"

### Реализация 10 страниц услуг
1. **services/painting.html** - Малярные работы
2. **services/rough_finish.html** - Черновая отделка
3. **services/flooring.html** - Финишные покрытия
4. **services/tiling.html** - Укладка плитки
5. **services/construction.html** - Строительные работы
6. **services/facade.html** - Фасадные работы
7. **services/paving.html** - Тротуарная плитка
8. **services/concrete.html** - Бетонные работы
9. **services/roofing.html** - Кровельные работы
10. **services/electricity_plumbing.html** - Электрика и сантехника

Каждая страница включает:
- Детальное описание услуги
- Список видов выполняемых работ
- Описание процесса работы
- Галерею примеров (3 изображения)
- Секцию преимуществ (4 карточки)
- Отзывы клиентов (2 отзыва)
